# panoramas-viewer

panoramas-viewer 是一个全景图浏览插件，你可以在现代浏览器上使用该插件预览 360 x 180 的全景图。

_本项目仅为学习用_

## 使用方法

依赖 
- `TypeScript`
- `ThreeJS`，参看官网怎么配置 `tsconfig.json`
